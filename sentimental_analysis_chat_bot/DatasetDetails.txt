Amazon Review Dataset
Electronics data

Dataset Details: The dataset contains 10,000 entries and 2 columns. 
Columns: An integer rating (e.g., 1-5), indicating the user's rating for a product.
ReviewText: Text containing the user's review of the product.

Missing Values: column has a few missing values (6 out of 10,000).
dataset is suitable for sentiment analysis using the overall rating as a proxy for sentiment and reviewText as the input text for analysis.

Dataset link:
https://cseweb.ucsd.edu/~jmcauley/datasets.html#amazon_reviews
